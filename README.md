## ðŸ›  Running this project locally

1. Clone this repository

   ```sh
   $git clone https://github.com/noszczykmichal/v2
   ```

2. Go into the repository

   ```sh
   $cd v2
   ```

3. Install dependencies

   ```sh
   $npm install
   ```

4. Start the development server

   ```sh
   $npm start
   ```
